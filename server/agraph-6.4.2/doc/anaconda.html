Anaconda

Anaconda® is a data science platform consisting of a package manager, an environment manager, a Python distribution, and a collection of open source packages. It is free and easy to install, and it offers free community support.

Packages are managed by the package management system conda.

AllegroGraph client

We provide a conda packaging of the AllegroGraph Python client.

To install this package with conda run: 

conda install -c franzinc agraph-python
